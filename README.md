# Lahjati


Lahjati is a machine learning project for Arabic dialect classification from speech signals. The system analyzes audio recordings and classifies them as **Gulf** or **Non-Gulf** dialects using acoustic features such as MFCCs, pitch, and spectral characteristics.

The project evaluates multiple machine learning and deep learning models, including Random Forest, AdaBoost, Logistic Regression, SVM, CNN, and RNN, to identify the most effective approach for dialect recognition.

## Features

* Arabic speech preprocessing and cleaning
* Acoustic feature extraction
* Gulf vs Non-Gulf dialect classification
* Comparative evaluation of multiple ML and DL models
* Performance analysis using Accuracy, Precision, Recall, and F1-score
